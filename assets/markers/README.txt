DROP YOUR NFT MARKER FILES HERE
================================

1. Go to: https://carnaux.github.io/NFT-Marker-Creator/
2. Upload your album cover image (JPEG, high contrast)
3. Download the ZIP and extract these 3 files:
   - cover.fset
   - cover.fset3
   - cover.iset
4. Drop all 3 files into THIS folder (assets/markers/)

The base name MUST be "cover" to match the URL in index.html:
  <a-nft url="assets/markers/cover" ...>

Tips for best tracking:
- Use a JPEG with lots of visual detail and contrast
- Avoid plain gradients or minimal/text-only covers
- The image used for training must be the same physical print you scan
